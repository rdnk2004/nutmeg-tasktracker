# Tech Stack & Documentation

## Overview
This project is a web application with a **Python/FastAPI** backend and a **HTML/JS/Tailwind** frontend.

## Tech Stack

### Backend
- **Language**: Python 3.x
- **Framework**: [FastAPI](https://fastapi.tiangolo.com/) - A modern, fast web framework for building APIs.
- **Server**: [Uvicorn](https://www.uvicorn.org/) - An ASGI web server implementation (standard for FastAPI, implied).

### Frontend
- **Structure**: HTML5
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (loaded via CDN)
- **Logic**: Vanilla JavaScript (`script.js`)
- **Design Features**: 
  - Glassmorphism effects (backdrop-blur)
  - Custom Tailwind configuration in-file
  - Gradient backgrounds

## Setup & Installation

### Prerequisites
- Python 3.7+ installed.

### Backend Setup
1.  Navigate to the backend directory:
    ```bash
    cd backend
    ```
2.  Create a virtual environment (if not already present):
    ```bash
    python -m venv venv
    ```
3.  Activate the virtual environment:
    - **Windows**: `venv\Scripts\activate`
    - **Mac/Linux**: `source venv/bin/activate`
4.  Install dependencies:
    ```bash
    pip install fastapi uvicorn
    ```

## Running the Application

### 1. Start the Backend Server
From the `backend` directory (with virtual environment activated):
```bash
uvicorn main:app --reload
```
- The API will be available at `http://127.0.0.1:8000`.
- Documentation (Swagger UI) is automatically generated at `http://127.0.0.1:8000/docs`.

### 2. Start the Frontend
Since the frontend uses standard HTML/JS, you can open it directly or serve it.

**Option A: Direct Open**
- Open `nutmeg/frontend/index.html` in your browser.

**Option B: Serve via Python (Recommended)**
From the `frontend` directory:
```bash
python -m http.server 5500
```
- Open `http://localhost:5500` in your browser.
